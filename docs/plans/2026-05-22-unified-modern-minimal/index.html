这版做了什么

  • 统一视觉系统：白底 + 浅冷灰二级面 + 安全橙 accent。原报告页的米黄/衬线/Mono 编辑风改成 modern-minimal 工具感，与首页同源。
  • 轮询页改造：原本只是个右上角小指示器，现在升级为屏幕中央的横向 3 状态 tracker + 实时分析日志 + 已用/预计耗时进度条，兑现架构 §4 的不变量。
  • 导航瘦身：顶部从「巡检 / 报告 / 班组 / 设置」四项砍到「巡检 / 报告」两项，回到 CLAUDE.md 的 3 步主张。
  • Wide(1920) 收紧：容器 max-width: 1240px，2 列布局 + 右侧 sticky 信息栏，避免大屏两侧空旷。
  • 断点重新校准：980 / 820 / 680 / 640 都做了局部调整；最重要的是 1100 这档桌面 CTA 不再悬浮。
  • 避坑：无紫色渐变、无 emoji 图标、无圆角卡 + 左竖条彩色模板；色温和 hierarchy 一致，安全橙仅用于主要 CTA、进度活动态、规范引言竖线、整改建议 callout 四处。
