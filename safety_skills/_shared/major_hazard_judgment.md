# 重大事故隐患判定（建质规〔2024〕5号）

## 判定依据

《房屋市政工程生产安全重大事故隐患判定标准（2024版）》，建质规〔2024〕5号，
住建部 2024-12-13 印发。下称"判定标准"。

判定标准列出的事项一旦命中，属于法定意义上的"重大事故隐患"，安全员有上报义务。
本字段（`is_major` / `major_basis`）的目的是把这类情形从普通隐患里挑出来红标显示。

## 输出字段

每条 `finding` 都包含两个字段：

- `is_major` (bool)：是否命中判定标准里的某一条
- `major_basis` (string)：命中时引用的具体条款；不命中或不确信留空

## 触发原则（**最重要**）

**宁可漏判，不可误判。** 这条铁律覆盖以下所有具体规则：

1. **只有当照片证据明确指向判定标准里某一条**时，才置 `is_major=true`
2. `severity=重大` **不是** `is_major=true` 的充分条件 —— 二者独立判断，
   "重大" 表达伤害后果严重，"is_major" 表达法定上报义务
3. **不允许编造条款号**。如果你不确信某条隐患是否落入判定标准、
   或不确信对应第几条，一律 `is_major=false` + `major_basis=""`
4. 管理类（方案审批、资质、培训记录）**只有当画面有明显信号**
   （如危险作业区无监护人员、未挂施工铭牌、无关人员进入危险区）时才触发
5. 不在下方触发要点内、但你**确信**属于判定标准重大隐患的情形，
   同样置 `is_major=true` 并引用对应条目

## `major_basis` 文本格式（命中时使用）

```
《房屋市政工程生产安全重大事故隐患判定标准（2024版）》建质规〔2024〕5号 第 X 条 类别 — 命中要点简述
```

例：

```
《房屋市政工程生产安全重大事故隐患判定标准（2024版）》建质规〔2024〕5号 第 6 条 高处作业 — 临边高度 ≥2m 无防护栏
```

不命中或不确信时：`is_major=false`、`major_basis=""`（**空串**，不要写"无" / "不适用"）。

## 重大事故隐患触发要点（按判定标准类别归纳）

照片中证据明确指向以下任一条件时，对应 `finding` 必须置 `is_major=true`
并按上述格式填写 `major_basis`。

> 注：以下数值阈值与表述按对该标准 2018–2024 演进的训练认知 + 行业公认值整理，
> **严格 verbatim 数值与表述以官方 docx 为准**。
> Phase 6 RAG 落地前，对数字不确信即留 `is_major=false`。

### 【施工安全管理】
- 危险性较大的分部分项工程（基坑、模板支撑、起重设备安拆、脚手架等）
  未按规定编制专项施工方案、或未按方案实施
  —— 画面信号：方案要求的支撑 / 防护体系缺失或与现场严重不符
- 危险作业区无监护人员明显在岗

### 【基坑、边坡工程】
- 开挖深度 ≥ 5m 的基坑未按要求支护 / 降水 / 监测
  （画面：边坡裸露开裂、支护体系明显缺失或失效）
- 支护结构变形、位移、开裂；周边堆载超出方案限值
- 基坑临边无 ≥1.2m 防护栏 + 警示标志

### 【模板工程及支撑体系】
- 高大模板支撑体系（搭设高度 ≥ 8m，或搭设跨度 ≥ 18m，
  或施工总荷载 ≥ 15kN/m²，或集中线荷载 ≥ 20kN/m）未按方案搭设
- 支撑系统立杆、剪刀撑、扫地杆、顶托明显缺失或松脱
- 满堂支模梁下加密杆与架体未形成可靠连接

### 【脚手架工程】
- 落地式扣件钢管脚手架 ≥ 24m / 悬挑式 ≥ 20m / 附着升降脚手架，存在以下任一：
  - 连墙件缺失或拉结失效
  - 剪刀撑、横向斜撑缺失
  - 立杆基础悬空、不均匀沉降
  - 外架与建筑结构间无防护

### 【起重机械及大型施工机械】
- 起重机械（塔吊、施工升降机、汽车吊等）安全装置失效或缺失：
  限位器、力矩限制器、超载保护、防坠器、防碰撞装置
- 多塔交叉作业未设防碰撞 / 未编安拆方案
- 附墙件、基础节、标准节明显缺陷或安装不到位

### 【高处作业防护】
- 临边、洞口（高度 ≥ 2m）无有效防护：1.2m 双横杆栏杆 / 盖板 / 安全立网缺失或失效
- 攀登作业（人字梯、直梯）违规使用 —— 人字梯使用高度 > 2m
- 悬空作业未设安全带挂点 / 未使用安全带
- 屋面、阳台、楼梯、电梯井口防护缺失

### 【施工临时用电】
- 未采用 TN-S 接零保护系统（三相五线 / 单相三线缺失）
- 配电箱（总配 / 分配 / 末级）无门、无防雨、漏电保护器缺失或失效；
  末级箱"一机一闸一漏一箱"被破坏
- 电缆架空 / 埋地不规范，明显裸露破损

### 【施工现场消防】
- 临时消防设施（灭火器、消防水源、应急照明、疏散通道）缺失或被遮挡
- 易燃易爆品（油漆、气瓶、聚苯板等）露天堆放、与火源 / 动火作业距离不足
- 动火作业现场无监护、无防火措施

### 【装配式建筑工程】
- 预制构件吊装无方案、未设临时支撑或支撑明显不足
- 构件就位后未连接、未灌浆即拆除临时支撑

### 【拆除工程】
- 未按拆除方案逐层、逐段、对称拆除
- 拆除区域无围挡 / 警戒，存在结构倾覆 / 坠物伤人风险

### 【有限空间作业】
- 进入有限空间（地下管廊、储罐、检查井等）作业，
  未见气体检测仪 / 通风设备 / 监护人

### 【安全防护用品】
- 高处或机械作业人员未佩戴安全帽 / 安全带 / 反光背心，
  且属于上方任一类别中的危险作业

## 反例（**严禁**这样做）

- ❌ 看到 `severity=重大` 就回填 `is_major=true`，不查触发要点
- ❌ 把本地 skill 清单 ID（如 `S06-A01`）当作判定标准条款号写进 `major_basis`
- ❌ 编造条款号（如把"第 6 条"写成"第 6.2.1 条"）
- ❌ 用"不适用 / 无 / N/A"等占位文本填 `major_basis` —— 不命中就用空串

记住：判定依据是**法律语言**，前端会按"判定依据 · {major_basis}"原样渲染给安全员看，
任何虚构都会被现场人员立刻识破，损害产品可信度。
