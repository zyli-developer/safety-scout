# S08 起重机械检查清单

## A. 塔式起重机

| 编号 | 检查项 | 优先级 | 视觉 | 依据 |
|------|--------|--------|------|------|
| S08-A01 | 塔吊基础是否完好、无沉降、无裂缝 | P0 | 中 | JGJ196 |
| S08-A02 | 塔身是否垂直，无明显倾斜 | P0 | 中 | JGJ196 |
| S08-A03 | 附墙装置是否完好（自由高度超规范必须附墙） | P0 | 中 | JGJ196 |
| S08-A04 | **限位装置（力矩、起重量、变幅、行走、高度）是否齐全有效** | P0 | 低 | JGJ196 |
| S08-A05 | 钢丝绳是否完好（无断丝、断股、严重磨损） | P0 | 中 | JGJ196 |
| S08-A06 | 吊钩是否有保险装置（防脱钩） | P0 | 中 | JGJ196 |
| S08-A07 | 塔吊司机室是否完好、视野无遮挡 | P1 | 中 | JGJ196 |
| S08-A08 | 多塔作业是否设置防碰撞措施 | P0 | 中 | JGJ196 |
| S08-A09 | **吊物下方是否有人员通行/作业** | P0 | 高 | JGJ196 |
| S08-A10 | 司索工、信号工是否在岗、有证作业 | P0 | 高 | JGJ196 |
| S08-A11 | 塔吊周边是否设置警戒区 | P1 | 高 | JGJ196 |

| S08-B05 | 停靠层防护门是否设置且能锁定 | P0 | 高 | JGJ215 |
| S08-B06 | 附墙装置是否完好 | P0 | 中 | JGJ215 |
| S08-B07 | 是否限载明示（人数和重量） | P1 | 高 | JGJ215 |
| S08-B08 | 司机是否持证上岗 | P0 | 中 | JGJ215 |

## D. 汽车吊 / 履带吊

| 编号 | 检查项 | 优先级 | 视觉 | 依据 |
|------|--------|--------|------|------|
| S08-D01 | 支腿是否完全打开，垫木垫板牢固 | P0 | 高 | GB6067 |
| S08-D02 | 作业半径内是否清场 | P0 | 高 | GB6067 |
| S08-D03 | 与架空电线距离是否符合安全距离 | P0 | 中 | GB6067 |
| S08-D04 | 吊钩、钢丝绳完好 | P0 | 中 | GB6067 |
| S08-D05 | 起吊重物上方/下方是否有人 | P0 | 高 | GB6067 |

## 必报项
- 吊物下方有人 → 重大
- 钢丝绳明显磨损/断丝 → 重大
- 限位装置缺失或失效 → 重大
- 升降机停靠平台无防护门 → 重大
- 物料提升机载人 → 重大
- 汽车吊支腿未打开/无垫板 → 重大
