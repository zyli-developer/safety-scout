# S09 中小型机械检查清单

## A. 切割机 / 砂轮机

| 编号 | 检查项 | 优先级 | 视觉 | 依据 |
|------|--------|--------|------|------|
| S09-A01 | **切割机/砂轮机是否设置防护罩** | P0 | 高 | JGJ33 |
| S09-A02 | 砂轮片是否完好，无裂纹 | P0 | 中 | JGJ33 |
| S09-A03 | 操作人员是否佩戴护目镜 | P0 | 高 | JGJ80 |
| S09-A04 | 切割火花方向是否有遮挡，避开人员、易燃物 | P0 | 高 | GB50720 |
| S09-A05 | 设备接地接零是否良好 | P0 | 中 | JGJ46 |

| 编号 | 检查项 | 优先级 | 视觉 | 依据 |
|------|--------|--------|------|------|
| S09-B01 | 设备防护罩是否齐全 | P0 | 高 | JGJ33 |
| S09-B02 | 钢筋作业区是否设置警戒线/围栏 | P1 | 高 | JGJ33 |
| S09-B03 | 操作人员是否佩戴手套（钢筋切断除外） | P1 | 中 | JGJ33 |
| S09-B04 | 钢筋堆放是否整齐、防滚动 | P1 | 高 | JGJ33 |
| S09-B05 | 调直机料筒、卷扬筒是否有防护 | P0 | 中 | JGJ33 |

## 必报项
- 切割机/砂轮机无防护罩 → 较大
- 钢筋设备无防护罩 → 较大
- 焊工无面罩、防护用品 → 较大
- 电焊接线柱裸露 → 较大
- 设备未接地 → 较大
