# S12 人员行为补充检查清单

> 本清单是对 L1 中 A 组（人员防护）的强化和扩展，重点关注**违章操作**和**作业行为**。

## A. 防护用品完整性（与 L1 互补）

| 编号 | 检查项 | 优先级 | 视觉 |
|------|--------|--------|------|
| S12-A01 | 多人作业时是否所有人都按要求穿戴防护用品 | P0 | 高 |
| S12-A02 | 安全帽颜色是否符合岗位区分（白=管理、黄=普工、红=电工等） | P2 | 高 |
| S12-A03 | 是否穿短裤、背心等不符合工服要求的衣物 | P1 | 高 |
| S12-A04 | 是否佩戴必要的手套（不同工种不同要求） | P1 | 中 |

## B. 违章操作识别

| 编号 | 检查项 | 优先级 | 视觉 |
|------|--------|--------|------|
| S12-B01 | **是否有人攀爬脚手架/模板架/钢筋上下** | P0 | 高 |
| S12-B02 | **是否有人在高处临边作业未系安全带** | P0 | 高 |
| S12-B03 | 是否有人在吊物下方通行或作业 | P0 | 高 |
| S12-B04 | 是否有人单手攀爬梯子 | P1 | 中 |
| S12-B05 | 是否有人在动火作业附近吸烟 | P0 | 高 |
| S12-B06 | 是否有人在配电箱前堆放材料/作业 | P0 | 高 |
| S12-B07 | 是否有非操作人员在机械作业区 | P0 | 高 |
| S12-B08 | 是否有人在阳台、临边坐着、躺着休息 | P1 | 高 |

## C. 作业姿势与位置

| 编号 | 检查项 | 优先级 | 视觉 |
|------|--------|--------|------|
| S12-C01 | 高处作业人员位置是否安全（避免重心外探） | P0 | 中 |
| S12-C02 | 群体作业是否保持安全距离（避免互相伤害） | P1 | 中 |
| S12-C03 | 焊接、切割作业下方/周边是否有人 | P0 | 高 |

## D. 特殊作业人员资质（视觉可见）

| 编号 | 检查项 | 优先级 | 视觉 |
|------|--------|--------|------|
| S12-D01 | 特种作业人员是否佩戴胸卡/标识（焊工、电工、起重工等） | P2 | 中 |
| S12-D02 | 信号工是否站在视野良好位置、使用对讲机 | P1 | 中 |

## 必报项
- 工人攀爬脚手架/模板架上下 → 重大
- 高处临边作业无安全带 → 重大
- 吊物下方有人 → 重大
- 动火作业附近吸烟 → 重大
- 多人作业有人未戴安全帽 → 较大

## 重要提示
人员行为类隐患**位置必须具体到画面中第几个人**（"画面左侧戴红色安全帽的工人""画面中部坐在临边的工人"），便于复核。
