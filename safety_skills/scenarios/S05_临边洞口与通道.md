# S05 高处作业-临边洞口与上下通道检查清单

> ⚠️ **高坠是建筑业死亡事故率第一的事故类型**，占工地总死亡率约 50%。
> 本清单是工地最重要的检查清单，**必须严格逐项核查**。

## A. 临边防护（高坠第一防线）

> 临边定义：作业面与下方落差 ≥2m 的所有边缘部位。

| 编号 | 检查项 | 优先级 | 视觉 | 规范依据 |
|------|--------|--------|------|---------|
| S05-A01 | **作业高度 ≥2m 的所有临边是否设置防护栏杆** | P0 | 高 | JGJ80 第 4.1.1 |
| S05-A02 | 防护栏杆上杆高度是否 ≥1.2m | P0 | 高 | JGJ80 第 4.1.2 |
| S05-A03 | 是否设置中间横杆（高度约 0.6m） | P0 | 高 | JGJ80 第 4.1.2 |
| S05-A04 | 是否设置挡脚板（高度 ≥180mm） | P0 | 高 | JGJ80 第 4.1.2 |
| S05-A05 | 防护栏杆是否牢固（无晃动、变形、缺失） | P0 | 高 | JGJ80 第 4.1.3 |
| S05-A06 | 屋面、阳台、外脚手架与建筑间隙处临边防护 | P0 | 高 | JGJ80 第 4.1 |
| S05-A07 | 基坑、土方临边是否设置防护栏 + 警戒线 | P0 | 高 | JGJ80 第 4.1.7 |
| S05-A08 | 阳台、雨棚等悬挑结构边缘是否封闭防护 | P0 | 高 | JGJ80 第 4.1 |
| S05-A09 | 卸料平台临边是否设置防护栏 + 限载牌 | P0 | 高 | JGJ80 第 6 |

### 判定要点
- A01：用尺度感判断 —— 一个成年人身高约 1.7m，落差超过 1 人身高即 ≥2m
- A02 / A03：常见违规 = 只有一道横杆（高度不足或缺少中横杆）
- A04：挡脚板常被遗漏，工具掉落时挡脚板可拦截
- A07：基坑临边 1m 内不应堆放材料

## B. 洞口防护

> 洞口定义：楼板、屋面、墙体上的开口。
> 根据洞口大小分类：

| 洞口尺寸 | 防护要求 |
|---------|---------|
| 25-500mm | 用盖板封闭 |
| 500-1500mm | 钢筋网 + 盖板 |
| ≥1500mm | 防护栏 + 安全网 |

| 编号 | 检查项 | 优先级 | 视觉 | 规范依据 |
|------|--------|--------|------|---------|
| S05-B01 | 边长 25-500mm 的小洞口是否用盖板封闭 | P0 | 高 | JGJ80 第 4.2.1 |
| S05-B02 | 边长 500-1500mm 的洞口是否用钢筋网或盖板封闭 | P0 | 高 | JGJ80 第 4.2.2 |
| S05-B03 | 边长 ≥1500mm 的洞口是否设置防护栏 + 安全网 | P0 | 高 | JGJ80 第 4.2.3 |
| S05-B04 | **盖板是否牢固固定（不允许只是搭一块板）** | P0 | 高 | JGJ80 第 4.2.4 |
| S05-B05 | 洞口是否有"小心坠落"等警示标识 | P1 | 高 | JGJ80 第 4.2.5 |
| S05-B06 | **电梯井口是否设置定型化防护门** | P0 | 高 | JGJ80 第 4.2.6 |
| S05-B07 | 电梯井内是否每隔两层（不超过 10m）设置一道安全平网 | P0 | 中 | JGJ80 第 4.2.7 |
| S05-B08 | 管道井、通风井等竖向洞口是否封闭 | P0 | 中 | JGJ80 第 4.2 |

### 判定要点
- B04：常见违规 = 盖板没固定，用脚一踢就翻 → 不合格
- B06：电梯井不能只用绳子拉一下，必须有定型化（焊接 / 装配式）防护门
- 露天的洞口最危险，重点排查

## C. 上下通道 ⚠️【人员通行核心】

> 严禁工人通过脚手架、模板架、井架等非专用通道上下楼层。
> 必须设置专用通道：**斜道 / 楼梯 / 爬梯 / 电梯**。

| 编号 | 检查项 | 优先级 | 视觉 | 规范依据 |
|------|--------|--------|------|---------|
| S05-C01 | **上下楼层是否设置专用通道** | P0 | 高 | JGJ80 第 6.0.1 |
| S05-C02 | **是否有人员攀爬脚手架/模板架/井架/绑扎钢筋等非专用通道**（严禁） | P0 | 高 | JGJ80 第 6.0.2 |
| S05-C03 | 斜道两侧是否设置防护栏杆 + 挡脚板 | P0 | 高 | JGJ80 第 6.0.3 |
| S05-C04 | 斜道板面是否设置防滑条（间距约 300mm） | P0 | 中 | JGJ80 第 6.0.4 |
| S05-C05 | 直爬梯是否两侧设置扶手或护笼（高度 ≥3m 时） | P0 | 中 | JGJ80 第 5.0.3 |
| S05-C06 | 移动梯子是否有防滑底脚，与地面夹角 60°-70° | P1 | 中 | JGJ80 第 5.0.5 |
| S05-C07 | 移动梯子顶端是否高出作业面 ≥1m | P1 | 中 | JGJ80 第 5.0.5 |
| S05-C08 | 楼梯口是否设置临边防护（上、中横杆 + 挡脚板） | P0 | 高 | JGJ80 第 4.1 |
| S05-C09 | 楼梯踏步是否完整，有无悬挑、缺损、湿滑 | P1 | 中 | JGJ80 第 6.0 |
| S05-C10 | **通道口（建筑出入口）上方是否设置防护棚（防物体打击）** | P0 | 高 | JGJ80 第 4.3.1 |
| S05-C11 | 防护棚是否双层、伸出长度足够（≥3m，悬挑长度 +1m） | P1 | 中 | JGJ80 第 4.3.2 |
| S05-C12 | 防护棚两侧是否封闭（防止侧向坠物） | P1 | 中 | JGJ80 第 4.3 |

### 判定要点 ⚠️
- **C01 / C02 是最高频违规**：工人为图方便，直接爬脚手架上下 → 极易坠落
- C03：斜道（俗称"马道"）必须有完整两侧防护
- C04：斜道无防滑条 → 雨天工人滑倒 → 高坠
- C10 / C11：通道口（人员进出口）必须有防护棚顶部封闭，防止上方坠物砸人

## D. 悬空作业

| 编号 | 检查项 | 优先级 | 视觉 |
|------|--------|--------|------|
| S05-D01 | 悬空作业人员是否系安全带且高挂低用 | P0 | 高 |
| S05-D02 | 悬空作业下方是否设置安全平网 | P0 | 高 |
| S05-D03 | 移动操作平台是否牢固、设防护栏、限载明示 | P0 | 高 |
| S05-D04 | 钢结构安装、网架施工等悬空作业是否设置安全绳挂点 | P0 | 中 |

### 判定要点
- D01："高挂低用"= 安全带挂点高于人腰部，作业时身体低于挂点
- 常见违规：安全带挂在脚手架水平杆上（错），应挂在垂直牢固结构上

| 编号 | 检查项 | 优先级 | 视觉 |

## 本场景"必报"项汇总

1. ✅ 临边高度 >2m 无防护栏 → severity: 重大
2. ✅ 防护栏杆缺中横杆/挡脚板/上杆不足 1.2m → severity: 较大
3. ✅ 洞口无任何防护或盖板未固定 → severity: 重大
4. ✅ 电梯井口无防护门 → severity: 重大
5. ✅ 工人通过非专用通道上下楼层 → severity: 重大
6. ✅ 斜道无防护栏 / 无防滑条 → severity: 较大
7. ✅ 建筑出入口无防护棚 → severity: 重大
8. ✅ 高处作业人员无安全带或安全带未挂牢固点 → severity: 重大
9. ✅ 悬空作业下方无安全网 → severity: 较大
