# L1 通用必查清单（每张图必查）

> **重要原则**：以下 35 项为工地最高频、最致命、视觉最易判断的隐患。
> 无论图片场景如何，**每一项都必须明确给出判断结果**：[存在 / 不存在 / 无法判断 + 依据]。
> **不可跳过任何一项**。判断为"不存在"也必须输出，证明你看过。

## A. 人员防护（高频项）

依据：《建筑施工高处作业安全技术规范》JGJ80-2016、《建设工程施工现场环境与卫生标准》JGJ146-2013

| 编号 | 检查项 | 优先级 | 视觉强度 |
|------|--------|--------|----------|
| A01 | 进入施工区域人员是否全部佩戴安全帽 | P0 | 高 |
| A02 | 安全帽佩戴是否规范（下颌带系扣、未反戴、未歪戴） | P1 | 高 |
| A03 | 人员是否穿着反光背心或工作服 | P1 | 高 |
| A04 | 是否穿合规劳保鞋（严禁拖鞋、凉鞋、赤脚） | P1 | 中 |
| A05 | 高处作业（≥2m）人员是否系安全带且高挂低用 | P0 | 高 |
| A06 | 特殊工种是否有对应防护：电焊作业-面罩+手套，切割作业-护目镜，粉尘作业-口罩 | P0 | 高 |
| A07 | 施工区域内是否有人员吸烟（非吸烟区） | P0 | 中 |

## B. 高坠风险（事故率第一）

依据：《建筑施工高处作业安全技术规范》JGJ80-2016

| 编号 | 检查项 | 优先级 | 视觉强度 |
|------|--------|--------|----------|
| B01 | 临边（落差>2m）是否设置防护栏杆 | P0 | 高 |
| B02 | 防护栏杆是否完整：上横杆 1.2m + 中横杆 0.6m + 挡脚板 ≥180mm | P0 | 高 |
| B03 | 楼层洞口、电梯井是否封闭或围护 | P0 | 高 |
| B04 | 楼梯口是否设置防护栏 + 挡脚板 | P0 | 高 |
| B05 | 预留洞口是否有盖板或防护标识 | P0 | 高 |
| B06 | 安全网（平网、密目网）是否完整、无破损、无脱落 | P0 | 高 |

## C. 用电安全（事故率第二）

依据：《施工现场临时用电安全技术规范》JGJ46-2005

| 编号 | 检查项 | 优先级 | 视觉强度 |
|------|--------|--------|----------|
| C01 | 配电箱是否关闭并上锁 | P0 | 高 |
| C02 | 配电箱周围 1.2m 范围内是否有杂物堆积 | P1 | 高 |
| C03 | 电线是否裸露、破皮、接头未绝缘包扎 | P0 | 高 |
| C04 | 电线是否拖地、浸水、被重物碾压、被钢筋穿挂 | P0 | 高 |
| C05 | 是否存在私拉乱接、一闸多机现象 | P0 | 中 |
| C06 | 移动电具是否使用合规防水插座/插头 | P1 | 中 |

## D. 物体打击与坍塌风险

依据：JGJ59-2011、JGJ130-2011

| 编号 | 检查项 | 优先级 | 视觉强度 |
|------|--------|--------|----------|
| D01 | 高处是否有未固定的物料、工具（易脱落坠物） | P0 | 高 |
| D02 | 材料堆放是否过高、过陡、有倾倒风险 | P1 | 高 |
| D03 | 脚手架是否有明显变形、缺杆、缺扣件 | P0 | 高 |
| D04 | 脚手架立杆底部是否落地稳固（无悬空、无垫砖、无松软基础） | P0 | 中 |
| D05 | 是否有未拆除的废弃支撑、模板、临时支架 | P1 | 中 |

## E. 消防安全

依据：《建设工程施工现场消防安全技术规范》GB50720-2011

| 编号 | 检查项 | 优先级 | 视觉强度 |
|------|--------|--------|----------|
| E01 | 动火作业现场是否配备灭火器（≥2具） | P0 | 高 |
| E02 | 动火作业 10m 范围内是否清理易燃物 | P0 | 高 |
| E03 | 易燃材料（泡沫板、油漆、保温棉）堆放是否远离火源、是否覆盖 | P0 | 高 |
| E04 | 气瓶（氧气、乙炔）间距是否 ≥5m，是否直立放置，是否远离火源 ≥10m | P0 | 高 |
| E05 | 消防通道是否被占用、堵塞 | P1 | 中 |

## F. 文明施工与通道

依据：JGJ146-2013、JGJ59-2011

| 编号 | 检查项 | 优先级 | 视觉强度 |
|------|--------|--------|----------|
| F01 | 施工通道是否通畅、无明显障碍物 | P1 | 高 |
| F02 | 是否有明显积水、油污、湿滑区域无警示 | P1 | 中 |
| F03 | 危险区域是否有警示标识、围挡（如基坑边、坠落区） | P0 | 高 |
| F04 | 现场是否有明显扬尘、未覆盖的散料 | P2 | 中 |
| F05 | 现场是否整洁，有无大量建筑垃圾混堆 | P2 | 中 |
| F06 | 关键部位是否缺少必要标识牌（如"必须戴安全帽""禁止吸烟"） | P2 | 高 |

---

## 判定原则

1. **疑罪从有**：拿不准的项标记为"无法判断"，并说明原因，**不要硬给结论**
2. **必须报告位置**：每条"存在隐患"必须指出在图片中的位置（如"图片左上区域""画面中部偏右"）
3. **必须给整改建议**：每条"存在隐患"必须给出具体可执行的整改措施
4. **必须标置信度**：高 / 中 / 低三档
